# Rural Routes Explorer PWA

Vercel-ready web/PWA conversion of the Rural Routes Xcode project.

## Deploy to Vercel

1. Upload/push these files to the GitHub repository connected to Vercel.
2. Vercel should detect Vite automatically.
3. Build command: `npm run build`
4. Output directory: `dist`
5. Visit your custom domain after deployment.

## Local preview

```bash
npm install
npm run dev
```

## Included

- Mobile-first traveler experience
- Towns, routes, side quests, passport progress
- QR/manual unlock support
- Collectible card galleries with front/back flip
- Turtle Trail, Fallen Badges, Women of the Border Queen, Gamblers & Gunfighters, Borderlands Legends
- Caldwell Mural Quest and handout image
- Local-storage progress persistence
- Basic local admin asset manager with import/export
- PWA manifest and service worker
